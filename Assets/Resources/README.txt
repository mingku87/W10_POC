=== 스프라이트 자동 로드 시스템 ===

이제 UI를 새로 생성할 때마다 매니저에 스프라이트를 다시 넣을 필요가 없습니다!
아래 폴더에 이미지 파일을 넣어두면 자동으로 로드됩니다.

📁 손님 스프라이트 (CustomerManager)
   위치: Assets/Resources/Sprites/Customers/
   
   필요한 파일:
   ✓ normal.png      - 멀쩡한 손님
   ✓ drunk.png       - 취한 손님  
   ✓ onphone.png     - 핸드폰 보는 손님

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

💡 사용 방법:
   1. 위 폴더에 이미지 파일 넣기
   2. UI 생성 (Generate All UI)
   3. Play 버튼 누르기
   → 자동으로 스프라이트가 로드됩니다!

📌 주의사항:
   - 파일 이름을 정확히 지켜야 합니다
   - 이미지 Import Settings에서 Texture Type을 Sprite로 설정하세요
   - 스프라이트가 없으면 기본 색상으로 표시됩니다
